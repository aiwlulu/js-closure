# js-closure

Static slides for JavaScript closure.

## Local Files

- `closure-slides.html`: main presentation page
- `index.html`: GitHub Pages entry that redirects to the presentation

## Deployment

This repository is set up for GitHub Pages publishing from the `main` branch root.

Repository settings:

- `Settings` -> `Pages`
- `Source` -> `Deploy from a branch`
- `Branch` -> `main`
- `Folder` -> `/(root)`

Expected site URL:

`https://aiwlulu.github.io/js-closure/`
